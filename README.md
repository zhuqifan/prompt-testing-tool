# Docker 部署说明

本项目已配置为使用 Docker 容器运行，history等数据存储在 SQLite 数据库中。

## Quick Start

### 使用 Docker Compose（推荐）

```bash
# 构建并启动容器
docker-compose up -d

# 查看日志
docker-compose logs -f

# 停止容器
docker-compose down
```
